# Telegram E-commerce Web App Сучасний магазин всередині Telegram (Mini App).

Technology Stack:

Frontend: HTML5, CSS3, JS (Telegram WebApp API).

Backend: Python, Aiogram 3.x.

Features:

Динамічна корзина.

Фільтрація за категоріями.

Haptic Feedback (вібрація).

Адаптивний UI (виглядає як нативний додаток).
